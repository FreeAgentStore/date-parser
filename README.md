# Date Parser

Parse any date or time string — 'next tuesday', 'March 14, 2025', 'in 3 days', '2025-03-14T10:00Z'. Evolved from 500+ real-world formats.

Part of [FreeAgentStore](https://freeagentstore.online) — AI tools that run free in your browser.

## Install

```bash
npm install @freeagentstore/date-parser
```

## Usage

```ts
export { parseDate, formatRelative } from "@freeagentstore/date-parser";
```

## ESM (zero install)

```html

  import { ... } from 'https://freeagentstore.online/pkg/date-parser/index.js'

```
